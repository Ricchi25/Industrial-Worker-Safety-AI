# Industrial-Worker-Safety-AI

An intelligent, computer-vision-powered safety assistant designed to monitor and enhance industrial workplace safety. Utilizing advanced AI models via Google AI Studio, this application detects Personal Protective Equipment (PPE) compliance, identifies hazardous situations, and helps maintain a secure environment for industrial personnel.

---

## 🚀 Key Features

* **PPE Detection:** Real-time checking for hard hats, safety vests, goggles, and steel-toe boots.
* **Hazard Identification:** Detects environmental risks such as spills, improper storage, or unauthorized zone entry.
* **Incident Reporting:** Automatically flags safety anomalies with visual snapshots and logs.
* **Interactive Dashboard:** Easy-to-use local interface built with Node.js to review safety streams and analytics.

---

## 🛠️ Tech Stack

* **Core Engine:** Gemini API via Google AI Studio
* **Frontend/Backend:** Node.js, Express, and modern JavaScript
* **Styling/UI:** Tailored interactive web dashboard
